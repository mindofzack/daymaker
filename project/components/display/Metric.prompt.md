Numeric data readout — the core marine building block (wind, waves, swell, SST). Tabular figures.

```jsx
<Metric label="Wind" value="12" unit="kt" sub="SE · gusts 18" />
<Metric label="Wave" value="2.4" unit="ft" trend={{dir:'up', value:'+0.6'}} tone="caution" size="lg" />
```

`tone`, `size` sm|md|lg, `icon`, `trend={dir,value}`, `sub`.
