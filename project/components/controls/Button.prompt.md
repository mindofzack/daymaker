Pill-shaped action button — use for any tap target; aqua `primary` is the singular accent, so one per view.

```jsx
<Button variant="primary" uppercase iconLeft={<PlusIcon/>}>Plan a trip</Button>
<Button variant="secondary">Save spot</Button>
<Button variant="outline" size="sm">Share</Button>
```

Variants: `primary` (aqua), `secondary` (dark), `outline`, `ghost`, `danger`. Sizes `sm | md | lg`. Set `uppercase` for the wide-tracked label voice. `fullWidth`, `disabled`, `iconLeft`, `iconRight`.
