Circular icon-only button — play/pause, nav arrows, overflow menus. Always pass an accessible `label`.

```jsx
<IconButton variant="accent" size="lg" label="Play"><PlayIcon/></IconButton>
<IconButton variant="surface" label="More"><MoreIcon/></IconButton>
```

Variants: `surface` (dark circle), `accent` (aqua), `ghost`, `outline`. Sizes `sm | md | lg`.
